# NKU-Treehole-Web

a H5 web frontend for NKU treehole.

## 介绍

南开树洞 APP H5 端

## 规划

- 从零使用 `react` 搭建系统
- 全面使用 `react-hooks` 开发
- ...

## 使用技术

- **UI 框架**: `react`、`react-hook`、`material-ui`
- **UI 组件**: `@material-ui`
- **状态管理**：`hox`
- **类型检查**：`typescript`
- **接口请求**：`axios`
- **cookies**：`js-cookie`
- **后端 API**：`Go`

## 运行

```bash
git clone git@github.com:psychopurp/nku-treehole-web.git
cd nku-treehole-web/
npm install
npm run start
```

## 启动 mock 测试

在/mock/user.json 中配置接口

```bash
npm run mockdev
```
